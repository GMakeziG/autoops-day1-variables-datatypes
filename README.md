# AutoOPS+ — Day 1 Mini-Project  
Variables & Data Types

This project includes:
- Bash, PowerShell, and Python examples  
- Global vs local variable demos  
- JSON and YAML parsing samples (`jq`, `yq`)  
- Data files for hands-on practice

```
└── data/

    └── config.yml

```
